## Installation

1. Download the APK file
2. Enable "Install from Unknown Sources" on your Android device
3. Install the APK
4. Grant required permissions (Overlay, Microphone, Accessibility)

## Requirements

- Android 8.0 (API 26) or higher
- 2GB RAM minimum (4GB recommended)

## Configuration

Before using, you need to configure API keys in the app settings:
- Tencent Cloud ASR (for voice recognition)
- MiniMax AI (for text polishing)

Contact the developer for test API keys.
