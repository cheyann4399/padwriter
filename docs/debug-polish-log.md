# AI 润色调试指南

## 快速开始

1. 连接手机到电脑，确保 ADB 可用
2. 运行日志监听命令：
   ```bash
   adb logcat -s StreamingPipeline
   ```
3. 手机正常使用悬浮球录音
4. 录音结束后，电脑终端显示润色流程日志

## 日志示例

```
=== AI 润色调试 ===
[ASR 原文]: 嗯我今天想说的是关于那个项目进度的问题
[后处理]: 我今天想说的是关于项目进度的问题
[词库替换]: 我今天想说的是关于项目进度的问题
[AI 润色]: 我今天想谈谈项目进度的问题。
==================
```

## 流程说明

| 阶段 | 说明 |
|------|------|
| ASR 原文 | 语音识别原始输出 |
| 后处理 | 去除语气词、重复内容 |
| 词库替换 | 根据自定义词库替换 |
| AI 润色 | MiniMax API 返回的最终结果 |

## 分支信息

- 分支名: `debug/polish-log`
- 改动文件: `StreamingPipeline.kt`
- 用途: 仅开发调试，不合并到 main
